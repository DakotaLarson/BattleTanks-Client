# BattleTanks Client

Project progress tracked on Trello: https://trello.com/b/XkG9gNRn

## Description

BattleTanks is a free 2.5d web based tank shooter where two teams of players battle it out in a team elimination gamemode!

## Controls
W, S - Movement

A, D - Rotation

Mouse - Aim/Shoot

E - Boost Speed

R - Reload

Esc - Open Game Menu

Enter - Open Chat
